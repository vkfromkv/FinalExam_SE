Project Instructions
================================

Instructions To view Table:

1. Extract zip file FinalExam_SE
2. Open terminal or command prompt at this level ../FinalExam_SE
3. In terminal >> cd final_exam_django

*Use python or python3 in according to your configuration
4. >> python manage.py flush
   select or type 'yes'

5. >> python manage.py import_courses
6. >> python manage.py runserver 9000
7. Open Browser
8. Type the address: "http://localhost:9000/courses/list/"
9. TABLE will be displayed

Instructions to run test file:
10. Come to terminal
11. exit server --> ctrl + c
12. Now in the terminal --> python manage.py test
